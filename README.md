# 📋 Мой список дел (Todo List)

Консольное приложение для управления задачами на Python.

## ✨ Возможности

- ✅ Добавление задач с датой создания
- 📋 Просмотр всех задач
- ✅ Отметка выполнения с датой завершения
- 🗑️ Удаление задач с пересборкой ID
- 🔍 Поиск по названию
- 🎯 Фильтрация (по статусу, дате, ID)
- 💾 Автосохранение в JSON

## 🚀 Установка и запуск

```bash
git clone <репозиторий>
cd Todolist-python
python main.py
```

или запуск /dist/main.exe

## 📦 Зависимости

Python 3.9+
Стандартные библиотеки (json, os, datetime и др.)
